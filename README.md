# Maina and Sons Car Hire | Premium Vehicle Rental Platform

A premium, responsive, multi-page front-end web application engineered for a luxury vehicle rental service. This application allows clients to explore an elite fleet, view rental rates, and book premium vehicles seamlessly across mobile, tablet, and desktop viewports.

## Project Details
- **Project Name:** Maina and Sons Car Hire Web Platform
- **Launch Date:** 15 June 2026
- **UI Architecture:** Bootstrap v5.3 (Local Offline Assets)
- **Core Technologies:** HTML5, CSS3, JavaScript (ES6)

## Key Features
- **Responsive Navigation:** Optimized sticky dark navbar with a mobile collapse hamburger menu.
- **Cinematic Hero Entry:** Modern, high-contrast entrance banner displaying business taglines and core actions.
- **Elite Fleet Gallery:** Dynamic presentation showcase grids displaying car classes, features, and rental pricing tiers.
- **Cross-Device Fluidity:** Mobile-first layout configurations built using the Bootstrap grid framework.

## How to Run the Platform Locally
1. Clone or download this project repository folder (`webapp-project`) to your local computer.
2. Open the project folder using **VS Code**.
3. Ensure you have the **Live Server** extension installed.
4. Right-click on `index.html` and select **Open with Live Server**.
5. The application will immediately serve on your local network port at `http://127.0.0`.
