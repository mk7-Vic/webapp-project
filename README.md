# Maina & Sons Car Hire

A modern, responsive multi-page car hire website developed using HTML5, CSS3, Bootstrap 5 and JavaScript.

The website showcases the services offered by Maina & Sons Car Hire, allowing customers to browse available vehicles, compare pricing, view locations, read customer reviews and make booking enquiries through an interactive contact form.

---

## Project Overview

Maina & Sons Car Hire is a fictional Kenyan car rental company created as a web development project.

The website demonstrates responsive web design principles, Bootstrap components, JavaScript interactivity and good user experience practices.
